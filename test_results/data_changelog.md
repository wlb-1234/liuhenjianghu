## 三、编码规则

### 3.1 代码结构
```
省级（2位）: 11-15, 21-23, 31-37, 41-46, 50-54, 61-65, 71, 81, 82
  ↓
市级（4位）: 前2位为省级代码 + 后2位序号
  ↓
区县（6位）: 前4位为市级代码 + 后2位序号
  ↓
街道（9位）: 前6位为区县代码 + 后3位序号
```

### 3.2 省级代码范围
```
11-15: 华北地区（北京、天津、河北、山西、内蒙古）
21-23: 东北地区（辽宁、吉林、黑龙江）
31-37: 华东地区（上海、江苏、浙江、安徽、福建、江西、山东）
41-46: 华中地区（河南、湖北、湖南、广东、广西、海南）
50-54: 西南地区（重庆、四川、贵州、云南、西藏）
61-65: 西北地区（陕西、甘肃、青海、宁夏、新疆）
71:    台湾省
81:    香港特别行政区
82:    澳门特别行政区
```

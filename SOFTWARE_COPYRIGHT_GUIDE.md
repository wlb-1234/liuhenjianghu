# 软件著作权申请材料指南

## 一、软著申请概述

### 申请机构
中国版权保护中心（CPCC）
- 官网：https://www.ccopyright.com.cn/

### 申请时间
- 普通申请：1-2个月
- 加急申请：5-10个工作日（需额外费用）

### 申请费用
- 普通申请：免费
- 加急申请：约 500-2000 元（根据时间）

## 三、源代码文档准备

### 格式要求
- 纸张：A4
- 页边距：上2.5cm，下2.5cm，左3cm，右2cm
- 字体：宋体，小四号
- 行距：1.5倍行距
- 页眉：软件名称和版本号
- 页脚：页码

### 内容要求
- 前30页：源代码开头部分
- 后30页：源代码结尾部分
- 每页不少于50行
- 总共约60页

### 源代码提取脚本

```bash
# 提取前端源代码
cd /workspace/projects/client
find . -name "*.tsx" -o -name "*.ts" | grep -v node_modules | head -100 > frontend_code.txt

# 提取后端源代码
cd /workspace/projects/server
find . -name "*.ts" | grep -v node_modules | head -100 > backend_code.txt
```

## 五、申请流程

### 步骤 1：注册账号
1. 访问 https://www.ccopyright.com.cn/
2. 注册账号
3. 完成实名认证

### 步骤 2：在线填报
1. 选择"计算机软件著作权登记"
2. 填写软件基本信息
3. 填写著作权人信息
4. 上传材料

### 步骤 3：打印材料
1. 打印申请表
2. 准备身份证明文件
3. 准备源代码文档
4. 准备软件说明书

### 步骤 4：提交材料
- 方式一：在线提交（推荐）
- 方式二：邮寄提交
- 方式三：现场提交

### 步骤 5：等待审核
- 普通申请：1-2个月
- 加急申请：5-10个工作日

### 步骤 6：领取证书
- 审核通过后，领取软件著作权登记证书

## 六、注意事项

1. **软件名称**
   - 必须与申请表一致
   - 建议使用"软件全称+版本号"格式

2. **版本号**
   - 首次申请使用 V1.0
   - 后续升级可申请变更登记

3. **开发完成日期**
   - 必须早于首次发表日期
   - 建议填写实际完成日期

4. **源代码**
   - 必须是完整的源代码
   - 不能包含注释掉的代码
   - 不能包含敏感信息（密码、密钥等）

5. **说明书**
   - 必须图文并茂
   - 必须包含软件界面截图
   - 必须说明主要功能

## 八、常见问题

### Q1：个人可以申请软著吗？
A：可以，个人和企业都可以申请软著。

### Q2：软著需要每年续费吗？
A：不需要，软著是终身有效的。

### Q3：软著申请被驳回怎么办？
A：可以根据驳回原因修改材料后重新提交。

### Q4：软著可以用于什么？
A：软著可以用于应用商店上架、项目申报、知识产权保护等。

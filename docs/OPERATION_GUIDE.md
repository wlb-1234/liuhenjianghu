## 二、ICP备案指南

### 什么是ICP备案

ICP备案是中国大陆网站的法定要求，所有在中国大陆提供服务的网站和App都需要进行备案。

### 备案前置条件

1. **域名**：需要一个已注册的域名（如 liuhenjianghu.com）
2. **服务器**：需要使用中国大陆的服务器（阿里云、腾讯云等）
3. **主体**：个人或企业备案

### 备案流程

```
1. 购买域名和服务器
   ├── 阿里云：https://www.aliyun.com
   ├── 腾讯云：https://cloud.tencent.com
   └── 华为云：https://www.huaweicloud.com

2. 申请幕布（个人备案需要）
   ├── 阿里云幕布申请：控制台 → 网站备案 → 幕布申请
   └── 顺丰快递到付

3. 填写备案信息
   ├── 主体信息（个人/企业）
   ├── 网站信息
   ├── 上传证件
   └── 幕布照片（个人）

4. 短信核验
   ├── 收到核验短信
   └── 24小时内完成验证

5. 管局审核
   ├── 审核时间：7-20个工作日
   └── 备案成功后会收到备案号
```

### 个人备案材料

| 材料 | 说明 |
|------|------|
| 身份证正反面照片 | 清晰可辨 |
| 幕布照片 | 站在幕布前拍摄 |
| 手机号 | 需接收验证码 |
| 邮箱 | 用于接收通知 |

### 企业备案材料

| 材料 | 说明 |
|------|------|
| 营业执照副本 | 复印件加盖公章 |
| 法人身份证正反面 | 复印件加盖公章 |
| 网站负责人身份证 | 正反面照片 |
| 幕布照片 | 网站负责人拍摄 |
| 公章 | 用于真实性核验单 |
| 法人手机号 | 需接收验证码 |

### 备案费用

| 项目 | 费用 |
|------|------|
| 域名注册 | ¥30-60/年 |
| 云服务器（最低配） | ¥30-60/月 |
| 备案服务 | 免费 |
| **总计** | **¥390-1020/年** |

### 快速备案方案

如果您想快速上线，可以考虑：

1. **使用已有备案的域名**
2. **使用二级域名**
3. **使用香港/海外服务器**（不需要备案，但国内访问速度慢）

## 三、软件著作权申请指南

### 什么是软件著作权

软件著作权（软著）是App上架应用宝、华为等国内应用商店的必备材料。

### 申请方式对比

| 方式 | 时间 | 费用 | 难度 |
|------|------|------|------|
| 自己申请 | 3-6个月 | 免费 | 复杂 |
| 代理机构 | 1-2个月 | ¥500-1500 | 简单 |
| 加急 | 15-30个工作日 | ¥1000-3000 | 简单 |

### 自己申请流程

#### 第一步：准备材料

1. **源代码文档**
   - 要求：60页以上，每页不少于50行
   - 格式：A4纸大小，页眉标注软著名称+版本号+页码
   - 内容：提供App的全部源代码

2. **软件说明书**
   - 要求：10页以上
   - 内容：
     - 产品介绍
     - 功能特点
     - 操作说明
     - 界面截图

3. **身份证明**
   - 个人：身份证正反面扫描件
   - 企业：营业执照+法人身份证

#### 第二步：网上登记

1. 访问中国版权保护中心官网：https://www.ccopyright.com.cn
2. 注册账号并登录
3. 填写软件著作权登记申请表
4. 上传申请材料
5. 确认并提交

#### 第三步：邮寄材料

1. 打印申请表
2. 准备源代码和说明书（纸质版）
3. 邮寄到中国版权保护中心

#### 第四步：等待审核

| 阶段 | 时间 |
|------|------|
| 初审 | 10个工作日 |
| 复审 | 20-30个工作日 |
| 公示 | 10个工作日 |
| 下证 | 通知领取或邮寄 |

### 代理申请推荐

| 代理机构 | 价格 | 时间 | 网址 |
|---------|------|------|------|
| 权大师 | ¥599起 | 1-2个月 | https://www.quandashi.com |
| 汇桔网 | ¥699起 | 1-2个月 | https://www.hujiang.com |
| 八戒知产 | ¥499起 | 1-2个月 | https://zbj.com |

### 软著证书样本

证书内容包括：
- 软件名称
- 版本号
- 著作权人
- 开发完成日期
- 权利范围
- 登记号
- 发证日期

## 四、各平台入驻指南

### 1. 苹果 App Store

**入驻地址**：https://developer.apple.com/programs/

**费用**：$99/年

**流程**：
```
1. 注册Apple ID
2. 加入Apple Developer Program
3. 填写开发者信息
4. 支付年费
5. 等待审核（1-2天）
```

**审核重点**：
- 隐私政策必须有且可访问
- 不得有令人反感的内容
- App功能需完整可用

### 2. 应用宝（腾讯）

**入驻地址**：https://open.qq.com/

**费用**：免费

**流程**：
```
1. 注册腾讯开放平台账号
2. 创建应用
3. 填写App信息
4. 上传材料（软著、ICP备案）
5. 提交审核（3-7个工作日）
```

**审核重点**：
- 软著必须与企业主体一致
- 隐私政策必须有效
- 内容不能违规

### 3. 华为应用市场

**入驻地址**：https://developer.huawei.com/

**费用**：免费（部分功能收费）

**流程**：
```
1. 注册华为开发者联盟账号
2. 实名认证（个人/企业）
3. 创建应用
4. 上传软著和备案
5. 提交审核（3-7个工作日）
```

### 4. 小米应用商店

**入驻地址**：https://dev.mi.com/

**费用**：免费

**流程**：
```
1. 注册小米开发者账号
2. 实名认证
3. 创建应用
4. 上传材料
5. 提交审核
```

### 5. 申请支付商户

#### 微信支付

**入驻地址**：https://pay.weixin.qq.com/

**条件**：
- 企业资质（个体工商户也可以）
- 营业执照
- 对公账户

**费用**：
- 费率：0.6%-1%

#### 支付宝

**入驻地址**：https://open.alipay.com/

## 六、快速上线方案

如果您希望快速上线但材料不全，可以采用以下方案：

### 方案A：先Web后App

```
Week 1-2：
├── 部署Web版本
├── 域名备案（同步进行）
└── 获取种子用户

Week 3-4：
├── 申请软著
├── 注册各平台账号
└── 准备App截图

Month 2-3：
├── 软著下证
├── 提交应用商店
└── APK构建完成
```

### 方案B：海外优先

```
Week 1-2：
├── 部署Web版本（国际节点）
├── 提交App Store（不需要软著和备案）
└── 推广获取海外用户

Month 3+：
├── 国内备案和软著
└── 提交国内应用商店
```

## 四、管理后台文件 (server/public/admin.html)

完整文件路径：`/workspace/projects/server/public/admin.html`

**核心功能：**
1. 登录表单：手机号 + 密码
2. 测试模式：前端直接验证 15613594588/admin123
3. Session 存储：localStorage.setItem('admin_session', ...)
4. 调试功能：Console 日志 + 清除按钮

**关键代码片段：**

```javascript
// 测试模式登录
if (phone === '15613594588' && password === 'admin123') {
  const testUser = {
    id: 999,
    phone: '15613594588',
    nickname: '管理员',
    member_level: 4,
    token: 'test_token_' + Date.now()
  };
  localStorage.setItem('admin_session', JSON.stringify(testUser));
  showAdminPage(testUser);
  initAdmin();
  return;
}

// 清除登录状态
localStorage.removeItem('admin_session'); location.reload();
```
